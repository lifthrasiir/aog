# Solver for the Artisan of Glimmith

This repository hosts a [Rust](https://rust-lang.org/)-based solver for [the Artisan of Glimmith](https://store.steampowered.com/app/4160210/The_Artisan_of_Glimmith/). It supports the full puzzle as described in [`GOALS.md`](./GOALS.md) which also serves as a concise description of the game.

All puzzles are described in an abridged description format, seen in `samples/`. `GOALS.md` should give all abbreviations for clues. The exact format is subject to change. Any first two solutions (if any) are shown by the solver as follows:

Two major solvers are provided: an edge-oriented [constraint satisfaction problem](https://en.wikipedia.org/wiki/Constraint_satisfaction_problem) solver and a shape-oriented [exact cover](https://en.wikipedia.org/wiki/Exact_cover) solver. There are also a host of specialized solvers invoked only when certain conditions are met, all in `src/solver/`.
